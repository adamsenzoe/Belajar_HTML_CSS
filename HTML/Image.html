tag < img > memiliki atribut :
src : untuk menentukan sumber file dan nama gambar,
width : untuk menentukan lebar gambar,
height : untuk menentukan tinggi gambar,
alt : untuk memberikan text alternatif jika gambar tidak dapat tampil,
title : untuk menambahkan text pada gambar, ketika mouse diatas gambar atau hover.
format gambar yang didukung oleh tag < img > adalah jpg, png, dan gif.
