tag < table > memiliki atribut :
cellspacing : untuk menentukan jarak antar cell,
cellpadding : untuk menentukan jarak isi sel dengan bordernya,
width : untuk menentukan lebar tabel,
height : untuk menentukan tinggi tabel,
bgolor : untuk memberi warna pada background tabel,
align : untuk menentukan perataan tabel,
border : untuk menentukan tebal border tabel.

tag < tr > dan < td > juga memiliki atribut :
align : untuk menentukan perataan text pada isi tabel,
valign : untuk menentukan perataan text secara vertical,
width : untuk menentukan lebar cell,
height : untuk menentukan tinggi cell,
bgcolor : untuk memberikan warna background pada cell,
rowspan : untuk menggabungkan sel baris berikutnya pada kolom yang sama,
colspan : untuk menggabungkan sel kolom berikutnya pada baris yang sama.

Contoh Tabel 1

Jadwal ujian

Hari Jam ke-1 Jam ke-2
Senin Bahasa indonesia Matematika
Selasa Bahasa inggris Kejuruan
